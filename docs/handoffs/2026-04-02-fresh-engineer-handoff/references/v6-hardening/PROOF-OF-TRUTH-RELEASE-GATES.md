# Proof-of-Truth Release Gates

## Gate 1: Golden path
One canonical route runs end-to-end without crash and persists all mandatory artifacts.

## Gate 2: Grounding
Grounding challenge set passes with deterministic verdicts.

## Gate 3: Budget
Budget admission and settlement work in exact and estimated modes with correct labeling.

## Gate 4: Safety
Unsafe commands, forbidden paths, and disallowed network/dependency changes are blocked or escalated.

## Gate 5: Patch truth
Keep/discard decisions match the expected outcomes on the patch challenge set.

## Gate 6: Read-model truth
Dashboard/read-model numbers can be traced to runtime artifacts with no fake derivations in production mode.

## Gate 7: Claim truth
Every externally stated product claim maps to a tested capability and approved wording.
