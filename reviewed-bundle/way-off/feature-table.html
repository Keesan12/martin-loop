CLI Commands Reference
martin init Initialize project. Creates martin.policy.yaml with defaults, checks API keys and environment.
martin doctor Health check: validates API keys, policy file schema, CVE status, and provider connectivity.
martin run Execute a governed loop. Enforces policy, leash, budget stops, and creates signed audit trail.
martin resume Resume from last verified checkpoint. Use after crash, timeout, or manual interruption.
martin explain Natural-language explanation of the last gate decision. Answers "why did Martin stop the loop?"
martin verify Verify Ed25519 signatures on the audit trail. Detects any post-run tampering.
