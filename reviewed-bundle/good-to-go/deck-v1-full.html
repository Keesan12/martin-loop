MartinLoop

A governed runtime for autonomous AI coding agents.

Budgets, policy checks, verifier gates, rollback evidence, and inspectable run records around the agent loop.

Problem

Agent loops need operational boundaries.

Cost

Loops can retry without knowing when the economics no longer make sense.

Safety

Commands, path scope, and secret-like task text need checks before work proceeds.

Evidence

Teams need records showing what happened and why the run stopped.
